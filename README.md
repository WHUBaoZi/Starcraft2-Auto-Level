# SC2 自动刷级脚本
星际争霸2国服自动刷经验脚本，需要双机协作
## 📦 安装配置
• 确保电脑安装了python
• 在脚本文件夹里，按住 Shift + 右键 → 选择"在此处打开命令窗口"
• 输入：pip install -r requirements.txt
## 🎮 使用方法
• 打开星际争霸2并运行ce修改器，ce下载及使用方式参见：https://www.bilibili.com/video/BV1h51vBfEKF
• 双人组队，进入合作模式界面，地图选择为残酷黑暗杀星
• 雷诺玩家运行Raynor.bat，另一玩家运行Others.bat
• 不要移动鼠标，等待自动刷级；15级后，威望会自动开启继续刷下一个威望；不想刷了关闭命令窗口即可
• 由于普残刷级较慢，推荐晚上双人挂机😞
